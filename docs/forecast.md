# Прогнозирование

## Методы прогнозирования:

С помощью данной библиотеки пользователи могут сформировать план для распределения товаров по региональным складам. В основе расчётов выходных параметров данного плана лежат две стратегии управления запасами:

### Метод 1: алгоритм на основе метода фиксированного размера заказа

Данная стратегия на основании ретроспективных данных о продажах позволяет сформировать план по контролю уровня запаса, не позволяя опускаться ниже критической границы, тем самым повышая риски out-of-stock. Данную стратегию предпочтительнее использовать селлерам, в связи с меньшей нагрузкой на склад.

В результате рассчитываются следующие параметры:
- страховой запас
- максимальный запас
- точка заказа 
- дата поставки 
- средний уровень запаса остатков на складе.

[Пример кода](https://github.com/strite-ru/supply-demo)

### Метод 2: алгоритм на основе метода фиксированного периода заказа

Данная стратегия на основании ретроспективных данных позволяет сформировать план по рассылке товаров по региональным складам для равномерного распределения нагрузки на склад. Пользователь вводит требуемый период для отправки партий товара, библиотека производит расчёт. 

В результате рассчитываются следующие параметры:
- страховой запас 
- максимальный запас 
- размер поставки 
- дата следующей поставки 
- средний уровень запаса.

[Пример кода](https://github.com/strite-ru/supply-demo)

### Метод 3: алгоритм на основе нейронной сети CatBoostRegressor

Модель была обучена на 600000+ дататочках и дает прогноз продаж на следующую неделю чтобы вы смогли максимально точно сформировать следующую поставку. 
При необходимости вы можете сделать прогноз на 2 недели.

#### Входные данные для CatBoost:

- цена продажи
- количество продаж
- месяц (1-12)
- день недели (0-6)
- продажи за вчерашний день
- продажи за последние 3 дня
- продажи за последние 7 дней

#### Выходные данные:

- количество продаж

Сделав более точной прогноз вы более точно формируете будущую поставку и снижаете затраты на хранение и логистику.

Модель дает наиболее точные предсказания если предоставить ей данные более чем 7000 отправлений.
